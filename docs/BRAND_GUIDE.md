### Icon Concept

The Hippo icon features a friendly, approachable hippo face with:
- **Round shape**: Represents memory storage (like a brain)
- **Purple gradient**: Tech-forward, creative, unique
- **Sparkles**: Memory/AI magic
- **Friendly eyes**: Approachable, trustworthy

## Usage Guidelines

### Do

- Use the purple gradient for brand recognition
- Keep the hippo friendly and approachable
- Maintain consistent spacing (8px grid)
- Use rounded corners (8px default)

### Don't

- Don't stretch or distort the logo
- Don't use non-brand colors for the icon
- Don't make the hippo look angry or scary
- Don't use low-contrast text
